Publicering

Nedan följer de steg som tagits för att optimera och publicera webbplatsen i en automatiserad webbutvecklingsmiljö.

Detta har automatiserats:

  • Minifiering av HTML, CSS och JavaScript har gjorts med NodeJs, NPM och Parcel.
  • Bilderna är tagna från pexels.com och har sparats i Photoshop i formatet JPEG. I HTML-kodens picture-tagg har kod för automatiserad konvertering till WebP lagts in tillsammans med kod för ändring av bildstorlek.
  • Bakåtkompabilitet har ordnats med koden "browserslist" i filen "package.json".
  • Kopiering av arbetsfiler till mappen "dist" för publicering har gjorts med kommandot npm run start/npm run build i Visual Studio Code.
  • En README.md-fil har skapats tillsammans med commits i Github.
  • Publicering av webbplatsen har gjords med Github och Netlify.
